# xxim-core-flutter

[![Pub](https://img.shields.io/pub/v/xxim_core_flutter.svg?style=flat-square)](https://pub.dev/packages/xxim_core_flutter)

支持Flutter6端开发。惺惺是一个100%开源社交平台，每个人都可以搭建自己的服务器，掌握数据的所有权。

## 初始化

     XXIMCore core = XXIMCore();
     core.init(
       params: Params(
         deviceModel: "",
         deviceId: "",
         osVersion: "",
         platform: "",
         appVersion: "",
         language: "",
       ),
       connectListener: ConnectListener(
         onConnecting: () {},
         onSuccess: () {},
         onClose: ({error}) {},
       ),
       receivePushListener: ReceivePushListener(
         onPushMsgDataList: (msgDataList) {},
         onFriendNotify: () {},
         onGroupNotify: () {},
         onSubscribeNotify: () {},
         onSocialNotify: () {},
       ),
     );

## 登录

     core.login(
       apiUrl: "",
       wsUrl: "",
       token: "",
       userId: "",
       networkUsed: "",
     );

## 登出

     core.logout();

## 是否登录

     core.isLogin();

## 修改语言

     core.setLanguage("");
